Version: 1.1.0-beta

  • Mechanics: Implemented quick-fill mechanic on barrels, with a configuration setting to disable the behavior. Use by right-clicking a barrel with an item containing fluid that can be put in the barrel. If the item can't fill the barrel for any reason, no fluid, wrong fluid, barrel is full, etc, then the block gui will be opened as normal.
  • Mechanics: Implemented code to make barrels compatible with automated fluid transport systems, such as those from the Buildcraft and Tubes mods. Fluids may be input to or extracted from any side of a barrel.
  • Mechanics: Reduced time to fill a barrel form a fluid container by 80 percent.
